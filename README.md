# 🎯 Jogo da Adivinhação

Um jogo simples onde o usuário precisa **adivinhar um número secreto entre 1 e 100**.  
O sistema informa se o número digitado é **maior ou menor** que o número secreto até que o jogador acerte.

Este projeto foi desenvolvido como parte de um **curso da Alura**, com o objetivo de praticar conceitos básicos de **HTML, CSS e JavaScript**.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎮 Como funciona

1. O jogo gera um **número secreto aleatório entre 1 e 100**.
2. O jogador digita um número no campo de entrada.
3. O sistema informa se o número secreto é:
   - maior 📈
   - menor 📉
4. O jogador continua tentando até **acertar o número**.
5. Ao acertar, o jogo mostra **quantas tentativas foram necessárias**.
